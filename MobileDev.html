Mobile Development
Semester 2

During semester two, I continued my work on my Mobile Development Module at Staffordshire University.

By the end of the module, I managed to produce a functional game that could be played on a mobile device or a device simulator. The game contained a suitable menu system and structure with transitions appropriate for a mobile game, included a monetisation strategy (with the use of ads), contained appropriate UI and is function for use on mobile devices, maintained an optimal framerate on a mobile device and made use of appropriate design patterns and programming practices, progression mechanics, local persistence of game data (a .json file saving and loading system for the players save file, as well as local persistence for the daily reward system), ad delivery system with both rewarded ads and interstitial ads, retention mechanics in the form of a daily reward system, and a native plugin for sharing to social media. Analytics were also included within the game.
