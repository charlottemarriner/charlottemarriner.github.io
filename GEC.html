Games Engine Creation
Super Mario Sistas

For semester two of my Games Engine Creation module, we were required to recreate Mario Bros. We did this by using SDL to make the game. Click here to access this project on Github.

The image above depicts the fifth level of my arcade Mario Bros remake. I decided that since we were looking originally at remaking this version through the tutorials, that after completing the first level, I would continue to make the next four levels. I downloaded the game on my Nintendo Switch so that I could play these levels and take a look at which enemies spawn and the contents of each level. Although some features of my game are a little different to the original, for the most part this is a attempted remake/reimagination of the original game. I'm very proud of this project and thoroughly enjoyed working on it - which is probably why it looks so much better when compared to the other project that was completed for this semester.
